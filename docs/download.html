TreeTag Downloads

Android

For an Android device that conforms to the requirements page:

  • TreeTag can be downloaded and installed from Google Play, here.
  • Alternately, it can be installed from an APK file: treetag_0.7.2.apk (25 MB)

Linux

On any 64-bit Linux OS, TreeTag can be installed using the script included in:

  • treetag_0.7.2.tar.gz (5 MB)

Windows

On a Windows system that conforms to the requirements page, all that is needed is:

  • treetag_0.7.2.zip (13 MB)

Alternately, TreeTag can be compiled from source — see the Installation page for more information.
